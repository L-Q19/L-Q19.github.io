正则表达式相关内容

发表于 2024-01-20

对于正则表达式相关内容的学习

元字符:具有固定含义的特殊符号

. ：匹配除换行符以外全部字符
\w:匹配所有数字、字母、下划线（如邮箱）
\s:任意空字符（使用少）
\d:匹配数字
\n:所有换行符
\t:匹配制表符

^:开启
$:结束（表达验证）

\W \S \D：类似反义词
a|b：或
():组
[ ]:字符组，支持[A-Za-z0-9]，[^]意为非

量词:控制元字符出现的次数

（）*：出现0次或更多次
（）+：出现一次或更多次
（）？：出现一次或者不出现
{n}：重复n次
{n,}：出现n次或更多次
{n,m}:出现n到m次

贪婪匹配和惰性匹配：
.*：贪婪匹配（最远的一项）
.*？：惰性匹配（最近的一项）

关于python数据可视化的学习——2022年冬

一

ndim 返回int 表示数组的维度
shape 返回tuple 表示数组的尺寸，n行m列为（n，m）
size 返回 int 表示数组的元素总数，扥与数字形状的乘积
dtype 返回data-type 描述数组中元素的类型
itemsize 返回int 表示数组的每个元素的大小（以字节为单位）

二 Numpy random

seed 种子
permutation 返回一个序列的随机排列
shuffle 对一个序列进行随机分布
binomial 产生二项分布随机数
normal 产生正态（高斯）分布的随机数
beta 产生beta分布的随机数
chisquare 产生卡方分布的随机数
gamma 产生gamma分布的随机数
uniform 产生在[0,1)中均匀分布的随机数

三 数组形态变换

reshape 修改数组维度
ravel flatten将数组展平
vstack 纵向拼接
hstack 横向拼接
hsplit(arr1,2) 纵向切割
vsplit(arr1,2) 横向切割
split（arr2，2，axis=1/0） 增加参数进行切割

四 Numpy进行统计

save 以二进制保存
load 以二进制读取
savez 以二进制保存多组数据
savetxt 保存为指定txt格式
loadtxt 读取为指定txt格式
# genfromtxt
sort 函数顺序排序axis=0为按列
argsort 顺序排序返回下标
tile 对数组进行复制
repeat 对数组元素进行复制
mean 统计数组所有元素的均值（axis=0/1统计列/行均值）
max/min（axis=0/1） 找出每列/行最大值/最小值
argmax（axis=0/1） 找出每列/行元素最大值
